Website Analytics Project

I built a website analytics dashboard for the leadership team, using data from Microsoft Clarity. Since there was no direct way to connect Clarity to Power BI, I automated data collection with Selenium and the Clarity API. The data was stored in Google Sheets and updated daily, which then fed into Power BI for clear, visual insights on website performance..


Purpose

The goal was to create a Power BI dashboard to track website performance and user behavior using Microsoft Clarity data.

Problem

The Analytics on Microsoft Clarity was not clear and it lacked a direct connection to Power BI, and so there was no easy way to get daily updates on website metrics.

Solution

  • I used Selenium to automatically download data from Clarity each day and store it in Google Sheets. I also used the Clarity API to fetch data directly. This data was then connected to Power BI for real-time reporting and insights.

Tools and Technologies Used

Microsoft Clarity Microsoft Clarity API Python Selenium Power BI Google Sheet
