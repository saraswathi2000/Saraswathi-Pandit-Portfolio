Marketing Analytics Solution

Automated the collection and analysis of marketing data from Facebook and Instagram for a digital marketing agency. Built a Power BI dashboard using Python, PostgreSQL, and Streamlit to deliver actionable insights, ensuring seamless updates and accessibility.


Purpose

To enable the client to monitor and optimize their Facebook and Instagram ad performance by automating data retrieval and visualization, ensuring timely insights for decision-making.

Problem

The Client faced challenges in manually gathering, managing, and analyzing data from Facebook and Instagram. This process was time-consuming and prone to errors, delaying critical insights for their marketing campaigns.

Solution

  • Automated data retrieval from Facebook and Instagram using their APIs with Python.
  • Stored the data in a PostgreSQL database hosted on Heroku, ensuring secure and scalable data management.
  • Built an interactive Power BI dashboard to visualize key metrics like ads reach, engagement,cost per lead..etc.
  • Automated dashboard updates using Python scripts to provide real-time data analysis.
  • Hosted the Power BI dashboard on a Streamlit application, offering seamless accessibility and collaboration.

Tools and Technologies Used


Facebook Graph API Instagram API Python PostgreSQL Power BI Streamlit Heroku
