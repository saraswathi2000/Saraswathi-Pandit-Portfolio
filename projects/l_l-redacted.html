Marketing Analytics Solution for Lion & Lion

Purpose

To enable Lion & Lion to monitor and optimize their Facebook and Instagram ad performance by automating data retrieval and visualization, ensuring timely insights for decision-making.

Problem

Lion & Lion faced challenges in manually gathering, managing, and analyzing data from Facebook and Instagram. This process was time-consuming and prone to errors, delaying critical insights for their marketing campaigns.

Solution

  • Automated data retrieval from Facebook and Instagram using their APIs with Python.
  • Stored the data in a PostgreSQL database hosted on Heroku, ensuring secure and scalable data management.
  • Built an interactive Power BI dashboard to visualize key metrics like ad reach, engagement, and ROI.
  • Automated dashboard updates using Python scripts to provide real-time data analysis.
  • Hosted the Power BI dashboard on a Streamlit application, offering seamless accessibility and collaboration.

Tools and Technologies Used

Python PostgreSQL (Heroku) Facebook Graph API Instagram Basic Display API Power BI Streamlit Heroku
